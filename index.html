Gerador QR-Code

"Para usar o QR code do Google Forms:

  1. Abra o leitor de QR code no celular.
  2. Aponte para o QR code e clique no link.
  3. Preencha o formulário e envie.

Para gerar o link no Google Forms:

  1. Crie o formulário.
  2. Clique em "Enviar" e copie o link.
  3. Converta o link em QR code.
  4. Compartilhe o QR code.
